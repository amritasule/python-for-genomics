# Practice Exercises

Test your Python skills with these genomics-focused exercises!

## How to Use

1. Open `exercises.py` and read the exercise descriptions
2. Try to solve each exercise on your own
3. When stuck or done, check `solutions.py` for answers
4. Each exercise builds on concepts from the tutorials

## Exercise List

### Data Structures
1. **Base Counter** - Count nucleotides using a dictionary
2. **Codon Extractor** - Split DNA into codons using lists
3. **Reverse Complement** - Create complement using string operations

### Conditionals
4. **Sequence Validator** - Check if DNA is valid
5. **GC Classifier** - Classify sequences by GC content
6. **ORF Checker** - Validate ORF structure

### Loops
7. **Pattern Finder** - Find all occurrences of a motif
8. **Sequence Comparer** - Calculate similarity percentage
9. **Multiple Frame ORF Finder** - Find ORFs in all frames

### File I/O
10. **FASTA Reader** - Read and parse FASTA files
11. **Sequence Filter** - Filter sequences by criteria
12. **FASTA Merger** - Combine multiple FASTA files

## Tips

- Start with the easier exercises (1-3)
- Read the exercise description carefully
- Test your code with the provided examples
- Don't peek at solutions until you've tried!
- It's okay to look at tutorials for help

## Running the Exercises

```bash
cd practice
python3 exercises.py
```
