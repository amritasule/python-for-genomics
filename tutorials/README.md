**File:** `02_lists_and_sequences.py`

**File:** `05_loops_and_iteration.py`

Iterate through sequences and data:
- For loops
- While loops
- Loop control (break, continue)
- Nested loops
- Enumerate and zip

### 6. File I/O in Genomics
**File:** `06_file_io_genomics.py`

Read and write genomic data files:
- Opening and closing files
- Reading files (read, readline, readlines)
- Writing files
- File modes (r, w, a)
